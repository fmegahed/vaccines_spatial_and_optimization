1 Objectives of this Document

This is the third Markdown in our analysis workflow (see step0_extract_and_eda.html and step0_transform.html for our previous analyses). The main objective of this document is to:

  • Fit a logistic growth model to describe the cumulative percent of vaccinated in a given county over time. Note that we limit the fits to the 2,397 counties with complete data, which are stored in the p_matrix_final.rds (generated by step0_transform.html in our previous analyses).
